## 📤 Deploy to Production (10 mins)

### Option 1: **Vercel (Recommended - Free, Easy)**

```bash
# 1. Make sure Git is set up
git add .
git commit -m "Upgrade portfolio to full-stack showcase"
git push origin main

# 2. Go to https://vercel.com
# 3. Click "New Project"
# 4. Select your GitHub repository
# 5. Click "Deploy"
# 6. Your site goes live! 🎉
```

**Your portfolio URL:** https://portfolio-website-[your-repo-name].vercel.app

### Option 2: **GitHub Pages (Free)**

```bash
# 1. Ensure repo is public
# 2. Go to Settings → Pages
# 3. Set source to "main" branch
# 4. Save

# Your site: https://[your-username].github.io/[repo-name]
```

### Option 3: **Netlify (Free)**

```bash
# 1. Connect GitHub
# 2. Select your repository
# 3. Deploy settings:
#    - Build command: (leave empty)
#    - Publish directory: (leave empty, root folder)
# 4. Deploy!
```

## 🚨 Common Issues & Solutions

### **Issue: Links not working**
**Solution:** Make sure you updated the actual URLs in `index.html`

### **Issue: Mobile layout broken**
**Solution:** Check viewport meta tag, zoom out in mobile view

### **Issue: API demo not working**
**Solution:** Check browser console (F12) for errors, ensure no CORS issues

### **Issue: Styles not loading**
**Solution:** Verify Font Awesome CDN link in HTML `<head>`

### **Issue: Hamburger menu not working**
**Solution:** Make sure JavaScript file loaded correctly
